Módulo I - Alfabeto

  • O que é um Alfabeto Manual?

    • É uma função da linguagem de sinais que usa as mãos para representar o alfabeto falado. Cada letra ou número é representado por uma configuração de mão específica. O alfabeto manual também é chamado de alfabeto numérico, reconhecimento de Alfabeto Digital, Datilologia ou Dactilologia.
